# Mari (mhr) iOS/macOS keyboards

## Mari iOS

#### Mari / Марий

`mhr-3-rows.yaml`

Standard Russian-based layout with Mari characters accessible via long-press:
- **о** → **ӧ**
- **н** → **ҥ**
- **у** → **ӱ**
- **ы** → **ӹ**
- **а** → **ӓ**

#### Mari (4-row) / Марий (4-рядная)

`mhr-4-rows.yaml`

Full layout with Mari letters and additional symbols on a dedicated top row.

### Versions sorting for iPhone:

    * mhr-3-rows.yaml
    * mhr-4-rows.yaml

## macOS

#### Mari / Марий

`mhr-macos.yaml`

Mari characters are available via the **Option (Alt)** layer:

*   **Option + а** = **ӓ** / **Option + Shift + а** = **Ӓ**
*   **Option + н** = **ҥ** / **Option + Shift + н** = **Ҥ**
*   **Option + о** = **ӧ** / **Option + Shift + о** = **Ӧ**
*   **Option + у** = **ӱ** / **Option + Shift + у** = **Ӱ**
*   **Option + ы** = **ӹ** / **Option + Shift + ы** = **Ӹ**
