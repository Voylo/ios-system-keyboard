**ISO 639-3**: `chv`

## Алфавит

`а ӑ б в г д е ё ӗ ж з и й к л м н о п р с ҫ т у ӳ ф х ц ч ш щ ъ ы ь э ю я`

**Встречаются только в заимствованиях**: `б г д ё ж з о ф ц щ ъ`
